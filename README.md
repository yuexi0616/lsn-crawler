功能介绍
本脚本为 LMS 在线实验平台作业页面增强工具，兼容油猴（Tampermonkey）和脚本猫（ScriptCat）。

核心功能
一键复制命令到剪贴板
实验报告中的每个代码块（<pre>、<code>）右上角自动添加「复制到终端」按钮。
点击后自动将命令复制到浏览器剪贴板，同时尝试自动写入平台「剪贴板」弹窗。
自动清理智能引号、全角标点、零宽字符，确保命令可直接在终端执行。
自动桥接平台剪贴板
自动查找并点击右侧栏「剪贴板」按钮，打开弹窗。
自动将命令写入弹窗文本框，并点击「确定」完成同步。
支持 Shadow DOM 和同源 iframe 内的元素查找。
使用方式
在脚本猫或油猴中安装本脚本。
打开 LMS 实验报告页面，代码块右上角会出现「复制到终端」按钮。
点击按钮，命令自动复制到剪贴板。
