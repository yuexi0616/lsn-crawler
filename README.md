## 快速开始

### Chrome 扩展（推荐）

无需手动登录 / 配置，一键操作，适合非技术用户：

1. 打开 Chrome 浏览器，访问 `chrome://extensions`；
2. 开启右上角「开发者模式」；
3. 点击「加载已解压的扩展程序」，选择项目内 `extension/` 目录；
4. 打开 LSN 实验作业页面，点击右下角浮动按钮 / 插件图标即可保存。

### 命令行脚本（Python/Node.js）

适合批量爬取场景，需基础命令行操作能力：

```bash
# Python 单页爬取示例
pip install requests
python crawl_LSN.py http://10.30.0.135/courses/194/assignments/2179?module_item_id=8601

# Node.js 批量爬取示例
npm install
node crawl_LSN.js --batch 2181 2184 8603 8606
```

## 详细使用指南

### Chrome 扩展 详细说明

#### 安装步骤

1. 确保 Chrome 浏览器版本 ≥ 80（低版本可能兼容异常）；
2. 进入扩展管理页：`chrome://extensions`；
3. 开启「开发者模式」（右上角开关）；
4. 点击「加载已解压的扩展程序」，选择本地项目的 `extension/` 文件夹；
5. 扩展安装成功后，浏览器右上角会出现插件图标。

#### 使用方式

| 操作场景     | 操作步骤                                                     |
| ------------ | ------------------------------------------------------------ |
| 保存当前页面 | 打开 LSN 实验 / 作业页面 → 点击右下角浮动按钮 ✨ 或插件图标 → 等待打包完成自动下载 |
| 批量爬取     | 点击插件图标 → 输入 Assignments 范围（如 2181-2184）和 Module ID 范围（如 8603-8606）→ 点击「开始爬取」 |

#### 输出格式

下载的 ZIP 压缩包解压后结构：

```Plain
[实验名称].zip
└── [实验名称]/
    ├── [实验名称].md  # 核心文档（图片为相对路径引用）
    └── [实验名称]_files/  # 图片资源目录
        ├── [实验名称]_img01.png
        ├── [实验名称]_img02.jpg
        └── ...
```

### Python 脚本 详细说明

#### 前置依赖

```bash
# 安装必要依赖（仅需首次执行）
pip install requests
```

#### 命令参数说明

| 命令格式                                          | 说明                                                         |
| ------------------------------------------------- | ------------------------------------------------------------ |
| `python crawl_LSN.py <URL>`                       | 单页面爬取：替换 `<URL>` 为 LSN 实验 / 作业页面的完整链接    |
| `python crawl_LSN.py --batch <A1> <A2> <M1> <M2>` | 批量爬取：A1/A2 为 Assignments 起止 ID，M1/M2 为 Module ID 起止 ID |

#### 可选配置（环境变量）

若爬取时提示权限不足，可配置账号密码：

```bash
# Windows
set LSN_USERNAME=你的LSN账号
set LSN_PASSWORD=你的LSN密码

# Mac/Linux
export LSN_USERNAME=你的LSN账号
export LSN_PASSWORD=你的LSN密码
```

## 功能特性

✅ **格式转换**：精准将 HTML 转为 Markdown（支持标题、列表、表格、代码块等）

✅ **代码高亮**：自动识别 15+ 种代码语言（Java/XML/Bash/SQL/Python/JS/Go 等）

✅ **图片处理**：自动下载页面图片，转为 Markdown 相对路径引用 

✅ **批量爬取**：支持指定 ID 范围批量爬取，带进度提示 

✅ **打包下载**：自动将文档 + 图片打包为 ZIP，便于归档 

✅ **多端支持**：Chrome 扩展（免配置）+ 命令行脚本（批量场景）

## 注意事项

1. 仅用于个人学习资料整理，请勿用于商业用途或违规爬取平台数据；
2. 爬取速度受网络 / 平台限制，批量爬取时请勿过快操作，避免触发平台风控；
3. Chrome 扩展仅支持「已解压目录加载」，请勿删除 / 移动 `extension/` 目录，否则扩展会失效；
4. 命令行脚本需保证网络可访问 LSN 内网（`10.30.0.135`），外网环境无法使用；
5. 若页面内容爬取不完整，可尝试刷新 LSN 页面后重新爬取。

## 常见问题

### Q1：Chrome 扩展安装后不显示浮动按钮？

A1：

- 确认已开启「开发者模式」且正确加载 `extension/` 目录；
- 刷新 LSN 实验页面（按 F5），浮动按钮通常在页面右下角；
- 检查 Chrome 扩展管理页，确认插件未被禁用。

### Q2：命令行脚本爬取提示 “403 Forbidden”？

A2：

- 配置 LSN 账号密码环境变量后重试；
- 确认 URL/ID 输入正确，且账号有权限访问该实验内容；
- 检查网络是否在 LSN 内网环境（如校园网 / VPN）。

### Q3：Markdown 图片无法显示？

A3：

- 确保图片文件保存在 `[实验名称]_files/` 目录下，且 Markdown 内路径为相对路径；
- 避免手动修改图片文件名 / 目录名，否则会导致路径失效。
